# 🌦️ Weather App

A simple and responsive Weather App built with **HTML**, **CSS**, and **JavaScript**. It uses the **OpenWeather API** to display real-time weather information for any city.

## ✨ Features

* 🔍 Search weather by city name
* 🌡️ Display temperature in Celsius
* 💧 Display humidity
* 💨 Display wind speed
* ☁️ Display current weather condition
* 🌤️ Weather icon based on current conditions
* ⌨️ Press **Enter** to search
* ⏳ Loading indicator while fetching data
* 🚫 Invalid city handling
* 🛡️ Error handling for failed requests

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* OpenWeather API

## 🚀 Getting Started

1. Clone this repository.
2. Create a free API key from OpenWeather.
3. Open `script.js`.
4. Replace:

```javascript
const apiKey = "YOUR_API_KEY";
```

with your own API key.

5. Open `index.html` in your browser.
